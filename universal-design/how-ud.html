Usability Heuristics

  • Check for erroneous typography, linkages, and spelling errors
  • Endorse proper coding standards through use of web validators
  • Ensure browser compatibility, always designing "mobile first"
  • Guarantee a suitable environment for screen reader interaction
  • Adhere to Outline structure denoted by HTML 5 Outliner (browser extension)
  • Incorporate CSS Website Layout to improve cross-platform usability
  • Adopt tabbed navigation principles for the benefit of motor-impaired individuals
  • Utilize alt text fields in order to increase assistance for visually impaired audiences
